Lần đầu (one-click):
  chmod +x setup.sh
  ./setup.sh

  Script sẽ tự động:
  1. Kiểm tra Docker đang chạy
  2. Khởi động PostgreSQL + tạo schema tự động (qua docker-entrypoint-initdb.d/)
  3. Chờ DB healthy → chạy generate_data_docker.py → ~15,000 bản ghi T3H
  4. Khởi động Flask web → http://localhost:5000

  Các lệnh khác:
  ./setup.sh generic   # Dữ liệu generic (không phải T3H)
  ./setup.sh tools     # Thêm pgAdmin tại http://localhost:8080
  ./setup.sh logs      # Xem logs realtime
  ./setup.sh reset     # Xóa sạch + chạy lại từ đầu
  ./setup.sh down      # Dừng tất cả

  Kết nối thủ công với psql/DBeaver:
  Host: localhost | Port: 5432
  DB: learning_data_system | User: postgres | Pass: postgres123

hệ).

quan hệ- đối tượng, CSDL phân tán, CSDL suy diễn, CSDL không quan hệ (NoSQL), CSDL không gian, CSDL
đa phương tiện.
- Thực hành: Học viên sẽ thiết kế các mô hình trên, thực hành các thao tác trên các hệ quản trị CSDL.
